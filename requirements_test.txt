# Test dependencies for the Fellow Stagg EKG+ integration.
#
# The pure protocol codec tests (tests/test_api.py) only need pytest + bleak.
# The Home Assistant-dependent tests (tests/test_config_flow.py) pull in
# Home Assistant via pytest-homeassistant-custom-component, plus the third-party
# requirements of the `bluetooth` and `usb` components that our integration
# transitively loads (these are not installed automatically by the test
# harness). If a future Home Assistant version changes those component
# dependencies, this list may need updating.

bleak
